## Dashboard CSS Variables

These colors are defined as variables in [index.css](file:///c:/Users/Biz/Documents/MPL%20Website%202%20-%20Edit/apps/web/src/index.css) and are primarily used in custom components (like custom date/time pickers):

*   **`--dash-primary`**: `#002442` (Dark navy tone)
*   **`--dash-secondary`**: `#fec330` (Bright gold yellow)
